# MikataApp-iOS 


## 1.Introduction
Name : 地方のミカタ
Description : Application for user who rent the house

### Requirements
* iOS 10.0+
* Xcode 9+


## 2.Installation

### Cocoapods

Install Cocoapods if need be.

```bash
$ gem install cocoapods
```

Then, run the following command.

```bash
$ pod install
```
### Carthage

Install Carthage if need be.

```bash
$ brew update
$ brew install carthage
```

Then, run the following command.

```bash
$ carthage update 
```
